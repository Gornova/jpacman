1. Introduction

Does really Pacman need an introduction? Eat everything, avoid or kill ghosts!
Make points, be the number one!


2. How to play


2.1 Windows system

Simply run jpacman.exe


2.2 Linux system

Give excutable permission to jpacman.sh, then run the script


3. Controls

Arrow - move pacman
F2    - fullscreen
